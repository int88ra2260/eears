# 培力英檢報名按鈕開關功能實現總結

## 功能需求
在後台的培力英檢管理頁面新增一個可以控制首頁"培力英檢報名"按鈕顯示與否的開關，以便控制報名時間。

---

## ✅ 已完成的修改

### 1. 後端 API 端點（settingsRouter.js）

#### 新增 GET 端點
**路徑：** `GET /api/settings/english-test-registration-enabled`

**功能：** 取得報名按鈕開關狀態

**回應格式：**
```json
{
  "enabled": true
}
```

**說明：**
- 不需要認證（公開端點）
- 預設值為 `true`（啟用）
- 如果資料庫中沒有設定，返回 `true`

---

#### 新增 PUT 端點
**路徑：** `PUT /api/settings/english-test-registration-enabled`

**功能：** 更新報名按鈕開關狀態

**請求格式：**
```json
{
  "enabled": false
}
```

**回應格式：**
```json
{
  "message": "設定已更新",
  "enabled": false
}
```

**說明：**
- 需要管理員權限（`authMiddleware` + `adminMiddleware`）
- 需要有效的 JWT token
- 只有 `role: 'admin'` 的使用者可以修改設定

**資料庫鍵值：** `english_test_registration_enabled`

---

### 2. 後台管理頁面（EnglishTestManagement.js）

#### 新增狀態管理
- `registrationEnabled` - 報名按鈕開關狀態（預設為 `true`）
- `isUpdatingSetting` - 更新設置中的狀態

#### 新增函數
1. **`loadRegistrationSetting()`** - 載入報名按鈕開關狀態
   - 在組件載入時自動調用
   - 從 API 讀取當前設定

2. **`handleToggleRegistration(enabled)`** - 更新報名按鈕開關狀態
   - 發送 PUT 請求更新設定
   - 更新本地狀態
   - 顯示成功/失敗訊息

#### 新增 UI 組件
在頁面頂部（標題下方）添加了一個卡片區域，包含：
- 標題：「首頁報名按鈕顯示控制」
- 說明文字：「控制首頁「培力英檢報名」按鈕的顯示與否，可用於控制報名時間」
- Bootstrap Switch 開關：
  - 啟用時顯示綠色「已啟用」
  - 停用時顯示灰色「已停用」
  - 更新中時禁用開關

**位置：** 在「培力英檢報名管理」標題和「匯出 Excel」按鈕下方

---

### 3. 首頁按鈕顯示邏輯（App.js）

#### 新增狀態管理
- `registrationEnabled` - 報名按鈕開關狀態（預設為 `true`）

#### 新增 useEffect Hook
- 在組件載入時自動從 API 讀取設定
- 錯誤處理：發生錯誤時保持預設值（啟用）

#### 修改按鈕顯示邏輯
**原邏輯：**
```javascript
const shouldShowEnglishTestButton = 
  location.pathname !== '/login' && 
  !location.pathname.startsWith('/admin');
```

**新邏輯：**
```javascript
const shouldShowEnglishTestButton = 
  registrationEnabled &&
  location.pathname !== '/login' && 
  !location.pathname.startsWith('/admin');
```

**說明：**
- 現在需要同時滿足兩個條件：
  1. 開關為啟用狀態（`registrationEnabled === true`）
  2. 不在 login 和 admin 頁面

## 🎯 使用方式

### 管理員操作步驟
1. 登入後台管理系統
2. 進入「培力英檢管理」頁面
3. 在頁面頂部找到「首頁報名按鈕顯示控制」卡片
4. 切換開關：
   - **啟用（綠色）：** 首頁顯示報名按鈕
   - **停用（灰色）：** 首頁隱藏報名按鈕
5. 設定會立即生效

### 用戶體驗
- **啟用時：** 用戶可以在首頁看到並點擊「培力英檢報名」按鈕
- **停用時：** 用戶在首頁看不到報名按鈕（即使滿足其他顯示條件）
