## ✅ 已完成的修改

### 1. 姓名驗證規則更新

**位置：** `EnglishTestRegistrationPage.js` 行 42-49

**原驗證規則：**
```javascript
const validateName = (name) => {
  const chinesePattern = /^[\u4e00-\u9fa5]+$/;
  if (!chinesePattern.test(name)) {
    return '姓名必須是中文';
  }
  return '';
};
```

**新驗證規則：**
```javascript
const validateName = (name) => {
  const chinesePattern = /^[\u4e00-\u9fa5]+$/;
  if (!chinesePattern.test(name)) {
    return '姓名必須是中文';
  }
  if (name.length < 2) {
    return '姓名必須至少兩個中文字以上';
  }
  return '';
};
```

**功能：**
- ✅ 驗證姓名必須是中文
- ✅ 驗證姓名長度至少兩個中文字以上
