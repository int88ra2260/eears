### ✅ 階段 1 & 2：按鈕顯示和導航修改

#### 修改 1：添加 useNavigate import
**位置：** `App.js` 行 2-8

**原代碼：**
```javascript
import {
  BrowserRouter as Router,
  Routes,
  Route,

```

```javascript
import {
  BrowserRouter as Router,
  Routes,
  Route,
  Navigate,
  useLocation,
  useNavigate  // ← 新增
} from 'react-router-dom';
```

**回滾方法：** 移除 `useNavigate` 即可

---

#### 修改 2：添加路由判斷邏輯
**位置：** `App.js` 行 58-62（約）

**新增代碼：**
```javascript
const navigate = useNavigate();

// 判斷是否應該顯示培力英檢報名按鈕（不在 login 和 admin 頁面顯示）
const shouldShowEnglishTestButton = 
  location.pathname !== '/login' && 
  !location.pathname.startsWith('/admin');
```

**回滾方法：** 刪除這兩段代碼即可

---

#### 修改 3：修改按鈕條件渲染和 onClick
**位置：** `App.js` 行 502-531（約）

**原代碼：**
```javascript
{/* 培力英檢報名按鈕 */}
<button
  onClick={() => setShowEnglishTestModal(true)}
  className="btn btn-lg"
  ...
>
  🎓 {isSmallMobile ? '培力英檢報名' : '培力英檢報名'}
</button>
```

**修改後：**
```javascript
{/* 培力英檢報名按鈕 - 只在非 login 和 admin 頁面顯示 */}
{shouldShowEnglishTestButton && (
<button
  onClick={() => navigate('/register/english-test')}
  className="btn btn-lg"
  ...
>
  🎓 {isSmallMobile ? '培力英檢報名' : '培力英檢報名'}
</button>
)}
```

**回滾方法：** 
1. 移除 `{shouldShowEnglishTestButton && (` 和對應的 `)}`
2. 將 `onClick={() => navigate('/register/english-test')}` 改回 `onClick={() => setShowEnglishTestModal(true)}`
