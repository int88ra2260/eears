### 變更摘要
移除所有硬編碼密鑰和敏感資訊，實施環境變數管理，強化系統安全性。

### 詳細說明
此 PR 解決了系統中的嚴重安全風險：
1. **硬編碼密碼問題**: 移除 `loginRouter.js` 中的明文密碼
2. **弱 JWT 密鑰**: 實施強密鑰管理
3. **資料庫密碼洩露**: 將資料庫認證資訊移至環境變數
4. **Email 認證資訊**: 移除硬編碼的 Gmail 帳號密碼

## 📝 額外說明

### 部署注意事項
1. **環境變數設定**: 必須先設定 `.env` 檔案
2. **密碼強度**: 建議使用強密碼
3. **JWT 密鑰**: 必須至少 32 個字元
4. **備份**: 部署前請備份現有配置

### 開發者指南
1. 複製 `env.example` 到 `.env`
2. 設定必要的環境變數
3. 執行 `npm install` 安裝新依賴
4. 執行測試確保功能正常

### 安全改進
1. **密碼加密**: 使用 bcrypt 加密密碼
2. **JWT 強化**: 使用強密鑰和適當的過期時間
3. **限流保護**: 實施登入和 API 限流
4. **安全標頭**: 設定安全相關的 HTTP 標頭
5. **輸入清理**: 清理惡意輸入
