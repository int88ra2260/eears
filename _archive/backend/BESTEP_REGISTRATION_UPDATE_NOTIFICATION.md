# 培力英檢報名資料修改通知功能

## 📧 功能說明

當學生成功修改培力英檢報名資料並送出後，系統會自動發送一封通知郵件到中心信箱 (`emicenter@mail.nsysu.edu.tw`)，以便行政端追蹤學生的報名資料修改情況。

## 🔧 實作細節

### 郵件模板

- **模板名稱**：`englishTestRegistrationUpdated`
- **收件人**：`emicenter@mail.nsysu.edu.tw`
- **發件人**：培力英檢郵件帳號（`BESTEP_GMAIL_USER`）

### 郵件內容

郵件包含以下資訊：

1. **學生資訊**
   - 姓名
   - 學號
   - 報名編號
   - 修改時間

2. **聯絡資訊**
   - 學生 Email
   - 學生電話

### 觸發時機

當學生通過以下 API 成功修改報名資料時會觸發：
- `PUT /api/english-test/registrations/update`

### 技術實作

1. **郵件發送位置**：`routes/englishTestRegistrationRouter.js`
2. **郵件模板定義**：`config/email.js`
3. **發送方式**：非同步發送，不阻塞 API 回應
4. **錯誤處理**：如果郵件發送失敗，會記錄錯誤但不影響報名資料更新

## 📋 郵件範例

```
【培力英檢報名資料修改通知】

有學生已成功修改培力英檢報名資料，請行政端進行追蹤。

【學生資訊】
• 姓名：王小明
• 學號：E12345678
• 報名編號：123
• 修改時間：2025-01-11 14:30

【聯絡資訊】
• 學生 Email：student@example.com
• 學生電話：0912345678

請至系統後台查看詳細修改內容。
```

## ✅ 測試建議

1. 使用學生帳號登入系統
2. 進入「查看與編輯」功能
3. 修改報名資料並送出
4. 檢查中心信箱 (`emicenter@mail.nsysu.edu.tw`) 是否收到通知郵件
5. 確認郵件內容包含學生姓名和報名編號

## 🔍 日誌記錄

系統會在以下情況記錄日誌：

- **成功發送**：`✅ 已發送報名修改通知郵件到中心信箱`
- **發送失敗**：`❌ 發送報名修改通知郵件失敗: [錯誤訊息]`

## ⚠️ 注意事項

1. 郵件發送是非同步的，不會影響報名資料更新的回應速度
2. 如果郵件發送失敗，報名資料更新仍然會成功
3. 確保 `BESTEP_GMAIL_USER` 和 `BESTEP_GMAIL_PASS` 環境變數已正確設定
4. 中心信箱地址固定為 `emicenter@mail.nsysu.edu.tw`
