# English Table 問卷更新日誌

## 更新日期：2024年12月

## 更新內容

### 1. 問卷結構變更
- **基本資料 Q3 變更**：從數值型「參加次數」改為分類型「本學期參與 ET 次數」
  - 選項：「5次以下」、「6-10次」、「11次以上」
- **李克特量表題目**：從 20 題縮減為 18 題
  - 移除舊版 Q4：「我能用英文解釋圖片或圖表的意思或主要訊息」
  - 移除舊版 Q16：「我會嘗試預測接下來可能會出現哪些問題或主題」
- **題目重新編號**：保持 q1-q18 的順序，確保 UI 邏輯清晰

### 2. 新題目內容（按順序）
1. 我能更流利地用英文談論我的個人經驗和意見
2. 我能對個人問題給出更詳細和有趣的回答
3. 我能更精準地使用英文描述圖表訊息或文章內容
4. 我能在活動討論中使用英文整理及表達個人想法
5. 活動進行中，我能使用生活例子來解釋及強化個人觀點
6. 我能將英文將課程主題與現實生活中的例子連結起來
7. 我現在比參加「ET」之前對說英文更有自信
8. 我說英文時比較不緊張或害怕犯錯
9. 我更願意在課堂上或在「ET」以外與他人用英文交談
10. 除修課規定外，我有個人理由參加「ET」活動，例如認識朋友、開口練習英文或者喜歡與他人互動！
11. 每天不同的主題和任務使我更有興趣參與
12. 我計劃未來繼續參加「ET」（或類似活動）
13. 我在討論中會用自己的話總結主要想法
14. 當我無法理解話題內容時，我會主動尋找桌長或其他人協助
15. 我會遵守輪流發言的規範，因為每個人都應該享有開口練習英文的機會
16. 這種由同儕主導、以討論為基礎的模式，幫助我更勇敢開口說英文
17. 「ET」氣氛佳且互動性強，使我說英文時感到很自在
18. 總體來說，「ET」對於提升我的英文口說技能是有益的

### 3. 技術實現

#### 後端變更
- **API 兼容性**：保持現有數據庫結構，舊問題 q19、q20 設為 null
- **驗證邏輯**：只驗證 q1-q18 為必填，允許舊問題為空
- **統計功能**：更新為只處理 q1-q18 的平均分數計算
- **Excel 匯出**：更新欄位標題，移除 q19、q20 欄位

#### 前端變更
- **動態問卷組件**：新增 `DynamicSurveyModal.js` 支援從 JSON 配置渲染問卷
- **問卷配置**：更新 `surveys.json` 為新的 18 題結構
- **管理員頁面**：更新題目對應表和圖表顯示
- **表單驗證**：支援新的分類型問題驗證

### 4. 數據庫兼容性
- **無破壞性變更**：保持現有 q1-q20 欄位結構
- **向後兼容**：舊數據仍可正常讀取和顯示
- **新數據處理**：新提交的問卷 q19、q20 自動設為 null
